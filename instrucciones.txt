INSTRUCCIONES DE USO

1. Crear el entorno virtual:
    python -m venv quantum_env

2. Activar el entorno:

    - Windows (PowerShell):
        .\quantum_env\Scripts\Activate.ps1

    - Linux/Mac:
        source quantum_env/bin/activate

3. Instalar dependencias:
    pip install -r requirements.txt


EJECUCIÓN DEL EXPERIMENTO:
    python experiment.py --mode --budget --repeats --out --agent --reward

PARÁMETROS:
    --mode <episodes|time>
        episodes → límite por número de episodios
        time     → límite por tiempo en segundos
    
    --budget <float>
        cantidad de episodios o segundos

    --repeats <int>
        número de repeticiones del experimento

    --out <ruta>
        carpeta donde guardar datos y gráficos

    --agent <quantum|classical>

    --reward <float>
        recompensa del entorno (default 10)
